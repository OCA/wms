Release channel plans allow you to organize your release channels inside a plan
to awake then all at once.

**Table of contents**

.. contents::
   :local:

Configuration
=============

In Inventory > Operations > Release Plans.

Only Stock Managers have write permissions.

Usage
=====

A typical use case would be to have a plan for each day of the week and at the
begin of the day, launch the plan of the day.
